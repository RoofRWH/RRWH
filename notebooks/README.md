# Public RRWH Notebooks

This folder contains the public notebook entry points for the RRWH workflow.

## Start here

1. `02_Data_Preprocessing_public.ipynb` - load CHIRPS data and compute rainfall statistics.
2. `RWH_distribution_fitting_public.ipynb` - fit the full public distribution set to AOI pixels.
3. `RWH_distributions_analysis_public.ipynb` - simplified rainfall distribution analysis notebook.
4. `07_Final_Policy_Outputs_public.ipynb` - generate district-level policy summary outputs.

## Notes

- These notebooks are public and generic.
- Update the paths in `config/paths.yaml` before running them.
- The notebook outputs are written to the folders defined in `config/paths.yaml`.
